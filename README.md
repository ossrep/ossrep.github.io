# ossrep.github.io

## Docs

The docs use Material for MkDocs https://squidfunk.github.io/mkdocs-material  
It uses a python venv  

Activate the venv
```shell
# Activate
source .venv/bin/activate
# Deactivate 
deactivate
```

To start locally
```shell
mkdocs serve
```
