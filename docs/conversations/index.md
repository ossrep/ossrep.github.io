## Adding New Conversations

When documenting a new conversation:

1. Create a new file with the naming convention: `YYYY-MM-DD-topic-slug.md`
2. Include the date, participants, and a brief summary at the top
3. Organize the content by topics discussed
4. Highlight key decisions and action items
5. Update this index page with a link to the new conversation
