# OSSREP - Open Source Retail Electric Provider

Welcome to the documentation for **OSSREP**, an open-source platform for managing a Retail Electric Provider (REP) in the state of Texas.

## Project Overview

OSSREP aims to provide a complete, production-ready platform for operating a retail electric provider in the ERCOT (Electric Reliability Council of Texas) market. The platform is built using modern, cloud-native technologies and follows microservices architecture principles.

## Technology Stack

| Layer | Technology |
|-------|------------|
| Backend | Quarkus (Java) |
| Frontend | Angular |
| Database | PostgreSQL |
| Documentation | MkDocs Material |

## Key Features

- **Customer Management** - Enrollment, account management, service addresses
- **Billing & Invoicing** - Usage calculation, invoice generation, payment processing
- **Usage/Metering** - Integration with Smart Meter Texas
- **ERCOT Integration** - Market transactions, enrollment, settlements
- **Customer Portal** - Self-service web application
- **Operations Portal** - Internal administration and customer service
